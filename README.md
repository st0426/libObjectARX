# libObjectARX

#### 介绍
libObjectARX是开源的ObjectARX通用函数库，支持从ObjectARX2007~ObjectARX2021，
Database，ADO，MfcGridCtrl不算，先放库里面，Excel里面的ExcelUTil也不是库文件，暂时放在里面
